# Аналитическое исследование рынка видеоигр (2000–2013)

Исследовательский проект, посвящённый анализу мирового рынка видеоигр за период 2000–2013 годов.

В ходе работы выполнены предобработка данных и исследовательский анализ, позволившие выявить наиболее популярные игровые платформы, жанры и особенности распределения продаж по регионам.

---

## Цель проекта

Цель проекта — выполнить предобработку данных и провести исследовательский анализ рынка видеоигр за период 2000–2013 гг., изучив популярность игровых платформ, жанров и распределение продаж по регионам.

---

## Используемые технологии

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Этапы исследования

- Загрузка и первичное изучение данных;
- Предобработка данных;
- Фильтрация датасета;
- Категоризация оценок пользователей и критиков;
- Исследовательский анализ данных;
- Формирование итоговых выводов.

---

## Основные результаты

В ходе исследования были изучены следующие вопросы:

- наиболее популярные игровые платформы;
- наиболее популярные жанры;
- жанры с наибольшим объёмом мировых продаж;
- распределение мировых продаж по регионам;
- распределение пользовательских и критических оценок игр.
